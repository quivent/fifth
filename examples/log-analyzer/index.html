Log Analyzer

Parse and summarize application logs. Pattern matching, SQLite aggregation, HTML reports.

Run

fifth examples/log-analyzer/main.fs /var/log/app.loganalyze log file

Features

  • Line-by-line — Memory-efficient processing
  • Pattern matching — Extract errors, warnings, metrics
  • SQL aggregation — Store in SQLite for querying
  • HTML reports — Visual summaries with charts

Why Fifth?

  • Static buffers fit line-at-a-time processing
  • SQL handles aggregation efficiently
  • HTML generation built-in
