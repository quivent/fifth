Static Site Generator

Generate a blog or documentation site from markdown files. Shell out to pandoc, output static HTML with consistent styling.

Run

fifth examples/static-site-generator/main.fs from project root

Features

  • Scan posts directory for markdown files
  • Convert to HTML via pandoc
  • Apply consistent templates and styling
  • Generate navigation and index pages
  • Output to dist/ directory

Structure

static-site-generator/
├── main.fs          # Entry point
├── templates.fs     # HTML templates
├── posts/           # Source markdown
│   └── example.md
└── dist/            # Generated output

Dependencies

pandoc

Why Fifth?

  • HTML DSL — Templating is natural with Forth words
  • No runtime — Static output works anywhere
  • Fast regeneration — Instant rebuilds
