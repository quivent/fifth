# Installation

## Homebrew (macOS)

```bash
brew tap quivent/fifth
brew install fifth
```

## From Source

```bash
git clone https://github.com/quivent/fifth.git
cd fifth && cd engine && make && cd ..
./fifth install.fs
```

Fifth installs itself to `/usr/local/bin`.
