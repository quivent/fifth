# Examples

Real-world Fifth programs.

## Generate an HTML Report

```forth
require ~/.fifth/lib/pkg.fs
use lib:core.fs

s" /tmp/report.html" w/o create-file throw html>file
s" Status Report" html-head html-body

s" System Status" h1.
<p> s" Generated by Fifth" text </p>

<table>
  <tr> s" Metric" th. s" Value" th. </tr>
  <tr> s" Users" td. s" 42" td. </tr>
  <tr> s" Uptime" td. s" 99.9%" td. </tr>
</table>

html-end
html-fid @ close-file throw
s" open /tmp/report.html" system
bye
```

## Query a Database

```forth
require ~/.fifth/lib/pkg.fs
use lib:core.fs

: show-users ( -- )
  s" users.db" s" SELECT name, email FROM users LIMIT 10" sql-exec
  sql-open
  begin sql-row? while
    dup 0> if
      2dup 0 sql-field type ." : "
      2dup 1 sql-field type cr
      2drop
    else 2drop then
  repeat 2drop
  sql-close ;

show-users
bye
```

## Dashboard with Stats

```forth
require ~/.fifth/lib/pkg.fs
use lib:core.fs
use lib:ui.fs

s" /tmp/dashboard.html" w/o create-file throw html>file
s" Dashboard" html-head ui-css html-body

grid-auto-begin
  42 s" Users" stat-card-n
  7 s" Active" stat-card-n
  99 s" Uptime %" stat-card-n
grid-end

html-end
html-fid @ close-file throw
s" open /tmp/dashboard.html" system
bye
```

## Command-Line Tool

```forth
\ wordcount.fs - Count words in a file

: count-words ( addr u -- n )
  0 >r  \ counter on return stack
  begin
    dup 0>
  while
    32 split-at    \ split on space
    2swap
    dup 0> if
      r> 1+ >r     \ increment counter
    then
    2drop
  repeat
  2drop r> ;

\ Usage: fifth wordcount.fs < file.txt
```
